# MANUAL INFORMAL DO GITHUB

1 - CLIQUE NO LINK ABAIXO, PARA ACESSAR A PÁGINA DO REPOSITÓRIO:
	https://github.com/ZeusBlima/ProjetoEscola

2 - NA PÁGINA DO REPOSITÓRIO, CLIQUE NO BOTÃO VERDE "CLONE OR DOWNLOAD", DEPOIS NO "OPEN IN DESKTOP"

3 - BAIXE E INSTALE A FERRAMENTA

4 - QUANDO O GITHUB DESKTOP ABRIR, CRIE SEU LOGIN, E ACESSE O SISTEMA (NÃO PULE ESSA PARTE)

5 - REPITA O PASSO 2. O NAVEGADOR VAI PERGUNTAR SE É PRA ABRIR O GITHUB DESKTOP. ACEITE

6 - O GH VAI PERGUNTAR ONDE É PRA CLONAR (BAIXAR) O REPOSITÓRIO. ESCOLHA O LOCAL E CONFIRME

7 - AGORA O GH VAI MOSTRAR DUAS VISÕES DO REPOSITÓRIO. 
	"CHANGES" SÃO AS MUDANÇAS QUE VOCÊ FEZ E AINDA NÃO FORAM COMPARTILHADAS.
	"HISTORY" SÃO TODAS AS ALTERAÇÕES QUE JÁ FORAM FEITAS.
	
8 - QUANDO VOCÊ ALTERAR OS ARQUIVOS, VAI APARECER EM "CHANGES" AS SUAS ALTERAÇÕES. 
	QUANDO VOCÊ TERMINAR UM CONJUNTO DE ALTERAÇÕES, INFORME O QUE VOCÊ FEZ EM "SUMARY" E CLIQUE EM COMMIT.
	QUANDO VOCÊ ESTIVER PRONTO PRA COMPARTILHAR AS INFORMAÇÕES, CLIQUE EM "SYNC", LÁ EM CIMA.
	QUANDO VOCÊ PASSAR ALGUNS DIAS SEM ALTERAR ARQUIVOS, ANTES DE COMEÇAR, CLIQUE EM "SYNC" TAMBÉM.

9 - PARA TESTAR SE ESTÁ TUDO FUNCIONANDO, ALTERE O ARQUIVO "README", NA RAIZ DO REPOSITÓRIO, COLOCANDO SEU NOME NA LISTA DE MEMBROS
	DEPOIS DE ALTERAR, "COMMIT" E "SYNC"
